to use this theme, add the base theme's css for your OS and your desired colorway's css to Tidaluna and activate both. 
<br>
## Instructions:
1.click the base theme file for your OS

![image](https://i.imgur.com/p3TqpGU.png)

2. click Raw button on top-right

![image](https://i.imgur.com/msp34oT.png)

3. copy link to the raw file's page

![image](https://i.imgur.com/0LWp1B2.png)

4. paste link in THEMES section of Tidaluna settings

![image](https://i.imgur.com/wxke59e.png)

5. repeat steps 2-5 but using the colorway's file instead of the OS base file

![image](https://i.imgur.com/acqgzy8.png)

6. if both the base theme and the colorway's theme are switched to active everything should work. you can store multiple colorways and switch between them if you like.
